Regex Data Extraction Project

This project was developed by Prince as part of the Regex Onboarding Hackathon. The goal is to extract specific data types from large text content using powerful Regular Expressions.

Extracted Data Types

  • Email Addresses (e.g. user@example.com)
  • URLs (e.g. https://example.com)
  • Phone Numbers (e.g. (123) 456-7890, 123-456-7890)
  • Credit Card Numbers (e.g. 1234 5678 9012 3456)
  • 12-hr and 24-hr Time Formats
  • HTML Tags
  • Hashtags
  • Currency Amounts

Technologies Used

  • Python (Regex Extraction Script)
  • HTML (Project Landing Page)
  • Git & GitHub (Version Control & Hosting)
